About Me

Professional Journey

I'm a Senior Data Engineer with extensive experience in building and maintaining large-scale data infrastructure on Google Cloud Platform. My expertise spans from designing ETL pipelines to implementing real-time data processing systems using Apache Beam and orchestrating workflows with Apache Airflow.

Technical Focus

I specialize in creating efficient, scalable data solutions that handle billions of records while maintaining data quality and reliability. My work involves optimizing query performance in BigQuery, designing stream processing architectures, and ensuring data governance across complex data ecosystems.

What I Work On

Data Pipeline Architecture

Designing and implementing end-to-end data pipelines using Apache Beam for both batch and streaming workloads, ensuring data consistency and fault tolerance across distributed systems.

Cloud Data Engineering

Leveraging Google Cloud Platform services including BigQuery, Cloud Storage, Dataflow, and Pub/Sub to build scalable and cost-effective data solutions.

Workflow Orchestration

Performance Optimization
